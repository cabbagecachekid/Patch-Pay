# ⌨️ Keyboard Shortcuts

## Route Calculator

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Enter` (⌘+Enter on Mac) | Calculate Routes | Instantly calculate routes without clicking |
| `Ctrl+K` (⌘+K on Mac) | Toggle Favorites | Show/hide your favorite routes |
| `Esc` | Close Panel | Close the favorites panel |

## Tips

- Shortcuts are shown below the Calculate button
- Works from anywhere in the Route Calculator
- Mac users see ⌘ instead of Ctrl
- Shortcuts don't interfere with browser shortcuts
